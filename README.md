# 建筑做法构造选型与自动出图软件

基于 **中南标图集(15ZJ001)** 的建筑构造做法选型与自动出图工具。

---

## 一、环境搭建

### 1. 安装 Python

需要 Python 3.8 或以上版本。安装时请勾选 "Add Python to PATH"。

- 官网下载: https://www.python.org/downloads/

### 2. 安装依赖

打开终端（CMD 或 PowerShell），进入软件目录：

```bash
cd C:\Users\user\building_software
pip install -r requirements.txt
```

依赖项：
| 库名 | 版本要求 | 用途 |
|------|---------|------|
| ezdxf | >=1.0.0 | DXF 图纸输出 (CAD格式) |
| reportlab | >=4.0.0 | PDF 图纸输出 |
| tkinter | (内置) | 图形用户界面 |

### 3. 字体说明

本软件使用 **仿宋_GB2312** 字体用于表格文字。Windows 系统通常已自带该字体。

如果系统没有该字体，请将 `simfang.ttf` 字体文件复制到 `C:\Windows\Fonts\` 目录，
或修改 `config/settings.json` 中的字体配置。

## 二、运行方式

### 启动程序

```bash
cd C:\Users\user\building_software
python main.py
```

### 操作流程

1. **项目信息设置** — 点击顶部 "项目信息设置" 按钮，填写：
   - 项目名称（必填）
   - 图纸名称、图号、设计人、日期、比例等

2. **选择建筑部位** — 在左侧面板下拉框中选择（地下室/住宅/公建）

3. **选择做法类型** — 选择五大类型之一（屋面/外墙/室内地面/室内墙面/室内天花）

4. **选择具体做法** — 在列表中选中一条或多条，点击 "添加选中做法 →"

5. **编辑构造层次** — 在中间面板可对上移/下移/编辑/新增/删除层次，点击 "保存修改"

6. **预览确认** — 在右侧预览面板中查看三张表的内容概览

7. **设置输出参数** — 顶部工具栏选择：
   - 图幅: A2 (594×420mm) 或 A3 (420×297mm)
   - 格式: DXF (CAD) 或 PDF
   - 输出目录: 可自定义

8. **一键生成** — 点击 "一键生成图纸" 按钮，生成三张表：
   - 构造做法表
   - 建筑装修一览表
   - 室内装修一览表

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销选择 |
| Ctrl+Y | 重做选择 |
| Ctrl+S | 生成图纸 |

---

## 三、输出文件

生成的图纸保存在 `output_files/` 目录下，文件名格式：

```
{项目名称}_构造做法表.dxf
{项目名称}_建筑装修一览表.dxf
{项目名称}_室内装修一览表.dxf
```

每张图纸包含：
- 标准图框（含装订边、标题栏）
- 项目信息（项目名称、图纸名称、图号等）
- 完整的表格数据

## 四、自定义做法库

### 做法库文件格式

做法库保存在 `config/methods_library.json`，结构如下：

```json
{
  "部位名称": {
    "做法类型名称": [
      {
        "id": "唯一编号（如 WM-ZZ-001）",
        "name": "做法名称",
        "reference": "参考图集编号",
        "layers": [
          {
            "order": 1,
            "material": "材料名称及做法描述",
            "thickness": "厚度（如 40，- 表示无厚度）",
            "note": "备注说明"
          }
        ]
      }
    ]
  }
}
```

### 添加新部位/做法

1. 直接在软件中使用 "导入做法库" 功能合并外部 JSON 文件
2. 或手动编辑 `config/methods_library.json`，按上述格式添加

### 部位与做法类型说明

默认支持的部位：
- **地下室**: 地下空间构造
- **住宅**: 住宅建筑构造
- **公建**: 公共建筑构造

每个部位下可配置五种做法类型：
- **屋面做法**: 自上而下（面层→结构层）
- **外墙做法**: 自外而内（饰面层→基层）
- **室内地面做法**: 自上而下（面层→基层）
- **室内墙面做法**: 自内而外（饰面层→基层）
- **室内天花做法**: 自上而下（基层→饰面层）

## 五、图框图块修改

图框参数配置在 `config/title_block.json` 中。

### 图框结构

```
┌─────────────────────────────────────────┐
│  外框 (细实线 0.13mm)                     │
│  ┌──────────────────────────────────┐   │
│  │  内框 (粗实线 0.4mm)              │   │
│  │  ← 装订边 25mm                    │   │
│  │                                  │   │
│  │       （表格绘制区域）              │   │
│  │                                  │   │
│  │                   ┌─────────────┐│   │
│  │                   │  标题栏      ││   │
│  │                   │  (右下角)    ││   │
│  │                   └─────────────┘│   │
│  └──────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 关键参数

```json
{
  "A2": {
    "paper_width": 594,       // 图幅宽度 mm
    "paper_height": 420,      // 图幅高度 mm
    "binding_margin": 25,     // 装订边宽度 mm
    "margin_left": 30,        // 内框左边距 mm
    "margin_right": 10,       // 内框右边距 mm
    "margin_top": 10,         // 内框上边距 mm
    "margin_bottom": 10,      // 内框下边距 mm
    "title_block": { ... }    // 标题栏布局
  }
}
```

### 修改标题栏

标题栏由 `title_block.rows` 数组定义，每行可含多列：

```json
"rows": [
  {
    "height": 8,              // 行高 mm
    "cols": [
      {"label": "设计号", "width": 45},  // 每列标签与宽度
      {"label": "图别", "width": 45}
    ]
  }
]
```

## 六、表格样式修改

表格样式配置在 `config/table_styles.json` 中。

可调整：
- 列宽（`width` 相对比例）
- 表头行高（`header_height` mm）
- 数据行高（`row_height` mm）
- 标题行高（`title_height` mm）
- 单元格内边距（`cell_padding.horizontal/vertical` mm）

---

## 七、全局设置

`config/settings.json` 包含：

| 设置项 | 说明 |
|--------|------|
| `line_weights` | 三级线宽: thick(粗)/medium(中)/thin(细) (mm) |
| `fonts` | 字号配置: title(表头)/header(列头)/body(数据) |
| `default_output_format` | 默认输出格式: "dxf" 或 "pdf" |
| `output_directory` | 默认输出目录 |

## 八、常见问题

**Q: 打开 CAD 后中文显示为乱码？**
A: 在 CAD 中设置文字样式为仿宋字体，或将系统仿宋字体文件复制到 CAD 字体目录。

**Q: DXF 线条粗细不明显？**
A: 在 CAD 中打开线宽显示（状态栏 LWT 按钮），或调整打印样式表。

**Q: 如何备份我的做法库？**
A: 使用软件中的 "保存做法库..." 功能另存为 JSON 文件，或直接复制 `config/methods_library.json`。

**Q: 生成的表格超出图框？**
A: 选择较少做法或使用 A2 图幅。软件会自动分页，超出的内容会生成额外图纸（文件名加 `_p2` 后缀）。
